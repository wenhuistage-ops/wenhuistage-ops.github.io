### B. 算法需求

#### 加班時數分類算法
```
輸入：dateType (例/日期), totalHours (總工時), isRestDay, isHoliday
輸出：overtimeDetails 各類型的時數分配

規則：
1. 先扣除正常工時（8小時或休息日時數）
2. 根據日期類型判斷加班倍率
3. 根據加班時數區間，分配到不同類別
```

#### 薪資計算邏輯
```
基本薪資 = 本薪
加班費 = Σ(各類型加班時數 × 時薪 × 倍率)
應發 = 基本薪資 + 例假日津貼 + 國定假日津貼 + 加班費
應扣 = 勳保費 + 健保費 + 住宿費 + 所得稅
實支 = 應發 - 應扣
```
