# 月曆格子顏色定義

本文檔描述月曆中各狀態格子的顏色定義，包括淺色模式和暗黑模式。

| 狀態 | 背景顏色 | 文字顏色 | 說明 |
|------|----------|----------|------|

## 其他樣式

- **holiday-text（假日文字）**：文字顏色為紅色，已在 `style.css` 和 `dist/compiled.css` 中定義
- **hover 效果**：所有格子在滑鼠懸停時會輕微上移並顯示陰影
