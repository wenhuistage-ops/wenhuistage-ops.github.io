### **1. 權限與安全隱患 [🔴 高優先級]**

#### **問題 1.1：管理員面板無身份驗證**
- **狀態**：✅ **已解決** (2026-04-14)
- **描述**：Admin Tab 現在每次點擊時都驗證真實權限
- **解決方案**：實現 `verifyAdminPermission()` 函數，每次操作都調用服務器 API
- **位置**：`js/core.js` 及 `js/app.js`
- **當前代碼**：
```javascript
tabAdminBtn.addEventListener('click', () => {
    const isUserAdmin = (localStorage.getItem("isAdmin") === 'true');
    if (isUserAdmin) {
        switchTab('admin-view');
    } else {
        showNotification(t("ERR_NO_PERMISSION"), "error");
    }
});
// ❌ 只檢查本地 localStorage，未驗證服務器
```

**改善方案**：
```javascript
// ✅ 每次點擊都驗證真實權限
tabAdminBtn.addEventListener('click', async () => {
    const res = await callApifetch({ action: 'checkAdminStatus' });
    if (!res.isAdmin) {
        showNotification(t("ERR_NO_PERMISSION"), "error");
        return;
    }
    switchTab('admin-view');
});
```

**修復優先度**：🔴 **立即** | **預計工時**：1 小時 | **責任人**：前端

#### **問題 4.2：缺少錯誤邊界（Error Boundary）**
- **狀態**：❌ **未實現**
- **描述**：API 調用缺少統一的錯誤處理
- **位置**：各 `.js` 文件中的 `callApifetch()` 調用
- **風險**：未捕獲的錯誤導致應用崩潰

**改善方案**：
```javascript
// ✅ 統一錯誤處理包裝器
async function callApiWithErrorHandling(params) {
  try {
    const response = await callApifetch(params);
    if (!response.ok) {
      showNotification(t(response.code || "API_ERROR"), "error");
      return null;
    }
    return response;
  } catch (err) {
    showNotification(t("NETWORK_ERROR"), "error");
    return null;
  }
}
```
